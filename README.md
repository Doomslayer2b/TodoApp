# TodoApp API

## Project Overview
- Minimal API built with .NET 8 and EF Core
- CRUD operations for Todo items + Categories
- Swagger documentation enabled

## Features
- Create, Read, Update, Delete Todos
- DTO validation ([Required], [MaxLength], [Range])
- Category 1:N relationship
- Error handling for invalid requests
- Fully tested endpoints
